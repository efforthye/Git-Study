## dev에서 작업 후 main에 추가하기
1. dev에서 작업 후 commit push
2. 깃허브 페이지의 pull requests에서 dev에서 개발한 것을 main으로 추가한다.
3. main으로 돌아와 fetch와 pull을 받는다.
4. dev로 돌아와 main을 기준으로 merge받는다.
5. 다시 작업한다.
